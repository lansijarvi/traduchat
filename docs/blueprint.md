# **App Name**: TraduChat

## Core Features:

- User Authentication: Secure sign-up/log-in via Google OAuth or email/password using Firebase Auth.
- Unique User Profiles: Users choose a unique username, creating a personal profile page (traduchat.app/@username).
- Real-time Messaging: Mobile-first chat UI with real-time message updates, typing indicators, and read receipts.
- Seamless Translation: Real-time translation of messages between English and Spanish using DeepL or Google Translate API.
- Language Selection Tool: Let the AI tool decide on preferred user language based on chat history. The user will be given a list of translation options to help inform the tool.
- Friend System: Users can search for and add friends to initiate chats, ensuring a clean, spam-free experience.
- Original Language View: A button to display the original, untranslated message.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trust and communication.
- Background color: Light blue (#E3F2FD) provides a gentle backdrop.
- Accent color: Warm orange (#FFB74D) highlights important interactive elements.
- Body text: 'Inter' (sans-serif) for a modern, neutral, and readable feel. Also good for headlines.
- Clean, simple icons to represent chat actions, friend requests, and language settings.
- Mobile-first responsive design for a native app-like feel on phones and tablets.
- Subtle animations for message delivery, typing indicators, and friend request updates.
